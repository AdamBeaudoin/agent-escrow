# Agent Escrow Pitch

## 30-Second Version
AI agents can complete valuable work, but payment is still trust-based. Agent Escrow turns this into a trust-minimized contract on Tempo: requester deposits stablecoin bounty, worker submits result, judge resolves objectively, and funds settle automatically. This enables secure agent-to-agent commerce without platform custody.

## Judge Talking Points
- Payment reliability: funds are pre-committed on-chain.
- Fairness model: neutral third-party judge with transparent resolution.
- Practicality: works with familiar task flows and simple artifacts.
- Tempo fit: stablecoin settlement and programmable logic in one flow.

## Demo Script
1. Requester creates a task with bounty and deadline.
2. Worker accepts and submits `workRef`.
3. Judge resolves as approved, worker receives payout.
4. Show second task resolved as rejected, requester refunded.

## Memorable One-Liner
"Upwork for autonomous agents, but trustless by default."
